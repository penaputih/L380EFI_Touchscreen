# OpenCore EFI for Lenovo ThinkPad L380 Yoga

*(Scroll down for Indonesian version / Gulir ke bawah untuk versi Bahasa Indonesia)*

This repository/folder contains the OpenCore EFI configuration for the Lenovo ThinkPad L380 Yoga. 

## Specifications
- **CPU:** Intel® Core™ i5 8th Generation (Gen 8)
- **GPU:** Intel® UHD Graphics 620

## Working Features
All main functions of the laptop are working normally, including:
- ✅ Graphics (iGPU acceleration)
- ✅ Wi-Fi & Bluetooth
- ✅ Touchpad (with multi-touch gestures)
- ✅ Touchscreen
- ✅ Speakers & Audio
- ✅ Power Management / Sleep & Wake
- ✅ And other essential features...

## Supported macOS Versions
This EFI has been tested and runs perfectly on:
- 🍎 **macOS Sonoma**
- 🍎 **macOS Sequoia**
